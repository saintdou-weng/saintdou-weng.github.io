✈️ Telegram Bot 設定
ℹ️ 設定後，Dashboard 的「✈️ Telegram」按鈕可直接推送報告到群組。群組成員輸入 /cctv 或 /patrol 也可呼叫 Bot 發送摘要。
🔗 GAS Web App URL （部署後取得）
💬 群組 Chat ID （負數，e.g. -1001234567890）
🔑 Secret Key （GAS setupProperties 裡設的 SECRET_KEY，防止外人偽造請求）
⚠ 此值只存在你自己的瀏覽器，其他人看不到。請與 GAS 的 SECRET_KEY 保持一致。
📋 GAS 部署步驟：
1. 開啟 Google Apps Script → 新增專案
2. 貼上 VRT_Security_Bot.gs 程式碼
3. 執行 setupProperties()，填入 Bot Token 和 Chat ID
4. 部署 → Web App → Anyone → 取得 URL 填入上方
5. 建立時間觸發器：onPoll → 每 1 分鐘
6. 群組輸入 /help 測試是否正常
